## 📌 Overview

The **Space Airline Management System** is a full-stack web application designed to simulate a modern airline management experience with a futuristic space-travel theme. The platform provides core airline features such as real-time flight tracking, ticket booking, seat reservation, and passenger record management.

Built using **Java**, **Spring Boot**, **React**, and **MongoDB**, this project demonstrates how a transportation management platform can be developed with a robust backend, interactive frontend, and dynamic database support.

***

## 🧩 System Design

The project follows a full-stack architecture:

- **Frontend (React):** Builds the visual dashboard, booking pages, seat selection screens, and management UI.
- **Backend (Spring Boot):** Handles business logic, booking operations, flight data processing, and passenger management.
- **Database (MongoDB):** Stores records for flights, tickets, seat reservations, and passenger details.
- **Deployment (Vercel):** Hosts the live version of the application.

## 🚀 Getting Started

### 1️⃣ Clone the repository
```bash
git clone https://github.com/AssassinjackM4/Space-Airline-App.git
cd Space-Airline-App
```

### 2️⃣ Set up the backend
- Install Java and configure Spring Boot.
- Set up MongoDB connection properties.
- Start the backend server.

### 3️⃣ Set up the frontend
```bash
cd frontend
npm install
npm run dev
```

### 4️⃣ Launch the project
- Ensure the backend server is running.
- Start the React frontend.
- Connect the application with MongoDB.
- Open the app in your browser and explore the platform.
